# Android Hiring App – Internship Project.                             ## Project Overview.                This is a simple Android application project created for hiring and mentoring Android development interns.

The goal of this project is to build a basic job/internship hiring app (similar to Apna / Indeed – MVP level).

## Features (Planned)
- User registration & login
- Job / internship listing
- Candidate application form
- Admin panel (basic)
- Firebase backend (authentication & database)

## Internship Objective
This project is intended for:
- Android development interns
- Final year students
- Freshers who want real-world project experience

Interns will work under guidance and learn:
- Android Studio.                      - Kotlin / Java.                      - Firebase integration.               - API handling                        ## Status                       Project setup & internship hiring phase.                               ## Company / Organizer       Registered business (GST available).
